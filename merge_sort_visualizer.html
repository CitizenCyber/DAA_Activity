Speed:

Divide, Conquer, Combine

Divide
Split the array recursively into two halves until each subarray contains one element.

Conquer
Sort each half recursively. Since a single element array is already sorted, recursion stops.

Combine
Merge two sorted halves together to produce a fully sorted array.

Recursion Tree

Time Complexity Analysis

Recurrence Relation

T(n) = 2T(n/2) + O(n)

Explanation:
  • 2T(n/2) → Recursively sorting two halves
  • O(n) → Merging two sorted halves
Using Master Theorem

T(n) = O(n log n)

Complexities
  • Best Case: O(n log n)
  • Average Case: O(n log n)
  • Worst Case: O(n log n)
